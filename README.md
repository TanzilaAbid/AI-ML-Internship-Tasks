## 📝 Detailed Project Summaries

### Task 1: Exploring and Visualizing a Simple Dataset
- **Objective:** Understand underlying patterns, perform data cleaning, handle missing entries, and use statistical plots to evaluate distributions.
- **Tools:** Python, Pandas, Matplotlib, Seaborn.

### Task 2: Predict Future Stock Prices
- **Objective:** Apply regression metrics onto historical financial data to forecast prospective valuation trends.
- **Evaluation:** Evaluated using standard continuous metrics like Mean Absolute Error (MAE) and Root Mean Squared Error (RMSE).

### Task 3: Heart Disease Prediction
- **Objective:** Build a medical classification system capable of analyzing patient biometrics (age, cholesterol, blood pressure) to determine cardiac illness risk.
- **Metrics:** Analyzed using Confusion Matrix, Accuracy Score, and Precision/Recall.

### Task 4: General Health Query Chatbot
- **Objective:** Implement a basic Natural Language Processing (NLP) conversational interface designed to parse user medical symptoms and map them to targeted health insights/remedies.

### Task 5: House Price Prediction
- **Objective:** Predict house values by analyzing features such as square footage, bedrooms, bathrooms, and asset condition.
- **Preprocessing:** Eliminated zero-value price entry anomalies and normalized structural scales using `StandardScaler`.
- **Model:** Trained utilizing an advanced **Gradient Boosting Regressor**.
- **Results:** Attained a benchmark MAE of $\approx \$162,962.34$ and RMSE of $\approx \$264,481.97$.
